**XML Unicode Checker** 

ist ein webbasiertes Tool, um XML-Dateien auf unerlaubte Unicode-Zeichen zu überprüfen. Hochgeladene XML-Dateien werden analysiert und verbotene Zeichen werden markiert, sodass Fehler leicht identifiziert werden können.


Funktionen

- Prüft XML-Dateien auf nicht erlaubte Unicode-Zeichen.
- Markiert verbotene Zeichen direkt in der Ausgabe (`>>>Zeichen<<<`).
- Zeigt den Pfad innerhalb des XML-Dokuments an, wo der Fehler auftritt.
- Unterstützt UTF-8 codierte XML-Dateien.
- Vollständig im Browser lauffähig (keine Installation nötig).
